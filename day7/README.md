# Day 7 - Jenkins CI/CD with Flask Hospital App

## Objective
To build and deploy a Flask Hospital application using Jenkins.

## Steps Performed
1. Opened Jenkins dashboard
2. Customized Jenkins configuration
3. Added credentials
4. Created and configured pipeline
5. Added Jenkinsfile
6. Pulled project files from repository
7. Built the pipeline
8. Created Flask Hospital application
9. Executed and verified the build process

## Result
Successfully configured Jenkins and built the Flask Hospital application using CI/CD pipeline.
